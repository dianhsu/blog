NeoVim配置

安装NeoVim

  • Windows: 从neovim github release下载最新的安装包，解压之后将目录添加到PATH即可
  • Linux: 可以通过包管理工具下载NeoVim，例如：sudo apt install neovim（Ubuntu）
  • macOS: 可以通过HomeBrew工具安装NeoVim，brew install neovim。
